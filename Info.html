About This Room Reservation System

This system is designed to manage classroom reservations efficiently for Polytechnic University of the Philippines - College of Engineering.

Purpose

  • ✔ Prevent room scheduling conflicts
  • ✔ Track reserved classrooms in real time
  • ✔ Display available rooms per selected date
  • ✔ Record who reserved the room

How To Use

  1. Select a room from the dropdown
  2. Choose a date
  3. Select start and end time
  4. Enter your name
  5. Click Reserve
